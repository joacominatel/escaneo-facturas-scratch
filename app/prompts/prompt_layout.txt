Tu trabajo es extraer los datos de manera **estructurada** de la siguiente factura escaneada. Devolvé la información en formato **JSON**, usando las mejores practicas como analista financiero, y completando la información de la forma mas detallada posible. Debes tener los siguientes puntos claves:

- invoice_number
- amount_total
- date
- bill_to # esto puede ser un objeto o un string, depende la cantidad de datos
- items: [
    {
      description: string,
      quantity: number | null,
      unit_price: number | null,
      amount: number | null

  ]
- currency
- payment_terms
